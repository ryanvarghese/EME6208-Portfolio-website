Marvel Universe Quiz

This project is an interactive Marvel Universe Quiz using drag-and-drop functionality. Users test their Marvel knowledge by sorting characters into appropriate team boxes (Avengers, X-Men, Guardians of the Galaxy, Defenders, and Wakandans). The quiz features a Marvel-themed design with custom colors, fonts, and background image. It combines HTML, CSS, and JavaScript to create an engaging, visually appealing, and educational game for Marvel fans.
